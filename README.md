# jszmq-test-page

This project is used to test [jszmq.js](https://github.com/zeromq/jszmq).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
